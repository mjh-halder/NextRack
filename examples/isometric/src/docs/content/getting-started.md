# Getting Started

Xyronos is a graphical modeling tool for designing data center infrastructure. Use the **System Designer** to place components on an isometric canvas, connect them, and configure their properties.

## Quick Start

1. Open the **System Designer** from the navigation rail (first icon).
2. Drag components from the **palette** on the left onto the canvas.
3. Click a placed component to open the **inspector** and edit its properties.
4. Connect components by dragging between their ports.

## Navigation

The left navigation rail provides access to different views:

| View | Purpose |
|------|---------|
| **System Designer** | Place and connect components on the isometric canvas |
| **Component Designer** | Create and customize component shapes |
| **Applications** | Manage application workloads |
| **Product Catalog** | Browse and manage hardware products |
| **Knowledge Base** | Documentation and reference (you are here) |
| **Admin** | Icon configuration and settings |

## Saving Your Work

Designs are saved automatically to your browser's local storage. Use the top header to name your design and manage save/load operations.
