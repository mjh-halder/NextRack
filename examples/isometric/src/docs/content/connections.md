# Connections

Connections represent physical or logical links between components — network cables, fiber optic runs, or power connections.

## Creating Connections

1. Hover over a component to reveal its **ports** (small circles on the edges).
2. Click and drag from a source port to a target port on another component.
3. Release to create the connection.

## Port Types

Ports define where and how components connect. Each port has:

- **Direction** — input, output, or bidirectional
- **Medium** — copper, fiber, or power
- **Bandwidth** — maximum data rate (e.g., 1G, 10G, 25G, 100G)

## Connection Properties

Select a connection (link) on the canvas to view and edit its properties in the inspector:

| Property | Description |
|----------|-------------|
| **Label** | Display name for the connection |
| **Medium** | Physical medium (copper, fiber) |
| **Bandwidth** | Link speed |
| **Encryption** | Whether the link is encrypted |

## Layout

Use the **auto-layout** feature in the toolbar to automatically arrange components and route connections for clarity.
