# Components

Components are the building blocks of your data center design. Each component represents a piece of infrastructure — a server, switch, firewall, storage array, or network interface card.

## Component Types

### Server
Rack-mounted compute units. Configure CPU, memory, and storage capacity.

### Switch
Network switches that connect servers and other devices. Define port counts and bandwidth.

### Firewall
Security appliances that control traffic between network segments.

### Storage
Dedicated storage arrays and NAS devices. Configure capacity and IOPS.

### NIC
Network interface cards installed in servers. Define bandwidth and medium (copper or fiber).

## Working with Components

### Placing Components
Drag a component from the palette onto the canvas. It snaps to the isometric grid automatically.

### Editing Properties
Click a component to open the inspector panel on the right. Each component type exposes different configurable properties.

### Connections
Components connect through **ports**. Drag from one port to another to create a link. The system validates that connections are compatible.

## Custom Components

Use the **Component Designer** to create custom shapes with configurable layers, icons, and port definitions. Custom components appear in the palette alongside built-in types.
